# 리스트
['수박','바나나','키위',1,2, True]


a = ['수박','바나나','키위',1,2, True]

# 모두 가능 
a[0]
a[-1]
a[3:]

```python
>>> a[0]
'수박'
>>> a[-1]
True
>>> a[3:]
[1, 2, True]
```

### 리스트 안에 리스트

b = [[1,2,3],[4,5,6],[7,8,9]]
b[0]
b[0][1]


```python
>>> b = [[1,2,3],[4,5,6],[7,8,9]]
>>> b[0]
[1, 2, 3]
>>> b[0][1]
```


c = ["문자열"]

```python
>>> c = ["문자열"]

'문'

'열'
```

### in 연산자 사용 -> 리턴값 -> true/false

>>> 1 in [1,2,3]
True
>>> 100 in [1,2,3]
